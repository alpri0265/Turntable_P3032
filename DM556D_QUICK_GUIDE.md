# Швидкий гайд: Підключення DM556D з STEP+/STEP- та DIR+/DIR-

## 🎯 Швидка відповідь

### До Arduino підключаємо:
```
STEP+ → Pin 6 (OUTPUT)
DIR+  → Pin 7 (OUTPUT)
```

### До купи (GND) з'єднуємо:
```
STEP- → GND
DIR-  → GND
GND   → GND (спільна земля)
VM-   → GND (від блоку живлення двигуна)
```

---

## 📋 Повна схема підключення

```
┌─────────────────────┐
│   Arduino           │
│                     │
│  Pin 6 (OUT) ───────┼──→ DM556D STEP+
│  Pin 7 (OUT) ───────┼──→ DM556D DIR+
│  GND ───────────────┼──→ DM556D STEP-  } Всі з'єднані
│  GND ───────────────┼──→ DM556D DIR-   } до купи
│  GND ───────────────┼──→ DM556D GND    } (GND)
│  5V ────────────────┼──→ DM556D VCC
└─────────────────────┘

┌─────────────────────┐
│  Блок живлення      │
│  12-24V DC          │
│                     │
│  + ─────────────────┼──→ DM556D VM+
│  - ─────────────────┼──→ DM556D VM- ──┐
└─────────────────────┘                  │
                                         │
                                         └──→ GND (спільна з Arduino)
```

---

## ✅ Правильне підключення

| Пін DM556D | Підключення | Куди |
|------------|-------------|------|
| **STEP+** | → | Arduino Pin 6 |
| **STEP-** | → | **GND (до купи)** |
| **DIR+** | → | Arduino Pin 7 |
| **DIR-** | → | **GND (до купи)** |
| **GND** | → | **GND (до купи)** |
| **VCC** | → | Arduino 5V |
| **VM+** | → | Блок живлення +12V до +24V |
| **VM-** | → | **GND (до купи)** |

---

## 🔌 Візуальна схема

```
                    ┌──────────────┐
                    │   Arduino    │
                    │              │
    Pin 6 ──────────┼──→ STEP+     │
    Pin 7 ──────────┼──→ DIR+      │
    GND ────────────┼──→ STEP- ────┼──┐
    GND ────────────┼──→ DIR- ──────┼──┤
    GND ────────────┼──→ GND ──────┼──┼──→ ВСЕ ДО КУПИ (GND)
    5V ─────────────┼──→ VCC       │  │
                    └──────────────┘  │
                                      │
    Блок живлення                     │
    +12-24V ───────────→ VM+          │
    GND ───────────────→ VM- ─────────┘
```

## ⚠️ Типові помилки

### ❌ НЕПРАВИЛЬНО:
```
STEP+ → 5V           ❌ НЕПРАВИЛЬНО!
STEP- → Pin 6        ❌ НЕПРАВИЛЬНО!
DIR+  → GND          ❌ НЕПРАВИЛЬНО!
DIR-  → Pin 7        ❌ НЕПРАВИЛЬНО!
```

### ✅ ПРАВИЛЬНО:
```
STEP+ → Pin 6        ✅ ПРАВИЛЬНО!
STEP- → GND          ✅ ПРАВИЛЬНО!
DIR+  → Pin 7        ✅ ПРАВИЛЬНО!
DIR-  → GND          ✅ ПРАВИЛЬНО!
```

---

## 📝 Чеклист підключення

- [ ] STEP+ підключено до Pin 6
- [ ] STEP- підключено до GND
- [ ] DIR+ підключено до Pin 7
- [ ] DIR- підключено до GND
- [ ] GND драйвера підключено до GND Arduino
- [ ] VCC підключено до 5V
- [ ] VM+ підключено до блоку живлення 12-24V
- [ ] VM- підключено до GND (спільна з Arduino)
- [ ] Всі GND з'єднані разом (до купи)

---

## 💡 Пояснення

**STEP+/STEP- та DIR+/DIR-** - це диференційні входи з оптопарами:
- **STEP+ та DIR+** - позитивні сигнали, підключаються до цифрових пінів
- **STEP- та DIR-** - негативні сигнали, завжди підключаються до GND
- Всі негативні піни (STEP-, DIR-, GND, VM-) з'єднуються до купи (GND)

---

**Готово! Всі піни з мінусом (-) з'єднуємо до купи (GND)!**
