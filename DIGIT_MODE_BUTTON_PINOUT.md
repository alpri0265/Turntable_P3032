# Розпіновка кнопки перемикання розрядів

## Підключення кнопки для перемикання розрядів

### Arduino Nano / Uno

```
Кнопка перемикання розрядів:
├── Вивід 1 → Arduino Pin 8
└── Вивід 2 → Arduino GND
```

### Arduino Mega 2560

```
Кнопка перемикання розрядів:
├── Вивід 1 → Arduino Pin 8
└── Вивід 2 → Arduino GND
```

---

## Технічні характеристики

- **Тип:** Тактовий вимикач (NO - нормально розімкнена)
- **Підключення:** INPUT_PULLUP (внутрішній підтягуючий резистор)
- **Пін:** 8 (для всіх платформ)
- **Debounce:** 500 мс (за замовчуванням)

---

## Функціональність

### Режими редагування розрядів:

1. **Одиниці (Units)** - зміна кута на ±1 градус
2. **Десятки (Tens)** - зміна кута на ±10 градусів
3. **Сотні (Hundreds)** - зміна кута на ±100 градусів

### Як використовувати:

1. Увійдіть в меню **"Set Angle"**
2. Натисніть кнопку перемикання розрядів для вибору режиму:
   - Перше натискання → **Units** (одиниці)
   - Друге натискання → **Tens** (десятки)
   - Третє натискання → **Hundreds** (сотні)
   - Четверте натискання → **Units** (знову одиниці, циклічно)
3. Обертайте енкодер для зміни кута відповідно до вибраного розряду
4. Натисніть кнопку енкодера для виходу з меню

---

## Приклад використання

### Швидке встановлення кута 270°:

1. Увійдіть в меню "Set Angle"
2. Натисніть кнопку розрядів → **Hundreds**
3. Обертайте енкодер → 200°
4. Натисніть кнопку розрядів → **Tens**
5. Обертайте енкодер → 270°
6. Натисніть кнопку енкодера для виходу

---

## Схема підключення

```
Arduino Pin 8 ──┬── Кнопка ──┬── GND
                │            │
         (INPUT_PULLUP)  (замикання)
```

**Примітка:** 
- Кнопка підключена між Pin 8 та GND
- Arduino використовує INPUT_PULLUP, тому кнопка працює на замикання (LOW при натисканні)
- Не потрібні зовнішні резистори

## Визначення в коді

В файлі `config.h`:

```cpp
// Кнопка перемикання розрядів (для введення кута вручну)
#if defined(ARDUINO_AVR_NANO) || defined(ARDUINO_AVR_UNO)
  #define DIGIT_MODE_BUTTON_PIN 8  // Пін для кнопки перемикання розрядів (Nano/Uno)
#else
  #define DIGIT_MODE_BUTTON_PIN 8  // Пін для кнопки перемикання розрядів (Mega)
#endif
```

---

## Перевірка підключення

1. Завантажте код на Arduino
2. Увійдіть в меню "Set Angle"
3. Натисніть кнопку перемикання розрядів
4. На екрані має змінитися режим (Units → Tens → Hundreds)
5. Обертайте енкодер - кут має змінюватися відповідно до вибраного розряду

---

## Усунення неполадок

### Кнопка не працює:
- Перевірте підключення до Pin 8 та GND
- Перевірте, чи кнопка працює (замикається при натисканні)
- Перевірте, чи ви знаходитесь в меню "Set Angle"

### Кнопка працює завжди:
- Перевірте, чи немає короткого замикання
- Перевірте, чи правильно підключено до GND

### Режим не змінюється:
- Перевірте, чи ви знаходитесь в меню "Set Angle"
- Перевірте debounce налаштування в `config.h`
